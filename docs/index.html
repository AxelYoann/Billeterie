Installation Rapide

Démarrez en quelques minutes

Windows

git clone https://github.com/your-username/multi-billeterie.git
cd multi-billeterie
setup.bat

Linux/macOS

git clone https://github.com/your-username/multi-billeterie.git
cd multi-billeterie
chmod +x setup.sh && ./setup.sh
Ou avec Node.js directement :
npm install
npm run setup
npm run dev
Prérequis : Node.js 18+, MongoDB, Git
